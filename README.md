# Library Book Manager (Python Mini Project)

This is my mini project for the Python course (Unit-2).  
The project is a simple Library Book Manager made using core Python concepts.  
It runs in the terminal and allows adding books, viewing, searching, borrowing,  
and returning books. I also added CSV file handling so the data is saved even  
after closing the program.

## üìå Project Features

### 1. Add Book  
I can add new books with ID, title, author, and number of copies.

### 2. View All Books  
Shows all the books in a table format.

### 3. Search Book  
Search by book ID or a part of the title.

### 4. Borrow Book  
Allows students to borrow a book if copies are available.  
The borrowed list is stored in a CSV file.

### 5. Return Book  
Students can return a book, and the stock updates.  
It also shows a list comprehension output of all borrowed books.

### 6. Data Storage  
The project uses:
- **books.csv** for storing all books  
- **borrowed.csv** for storing borrowed records  

This makes the project work even after restarting.

### 7. Menu Driven System  
The whole program runs inside a while loop until the user exits.
